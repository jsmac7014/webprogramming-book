# HTML이란

HTML은 하이퍼텍스트 마크업 언어(HyperText Markup Language)입니다. 이 HTML은 웹 페이지를 구성하는 요소이며 HTML은 제목, 단락, 목록등 문서에 사용되는 구조적 요소나 이미지, 비디오등 멀티미디어등을 지원합니다.

아래에 가장 기본적인 HTML의 예시가 있습니다.

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My Page Title</title>
  </head>
  <body>
    <h1>My Head</h1>
    <p>My paragraph</p>
  </body>
</html>
```
